2025년 2월 Claude Code 출시 이전까지 한 자릿수에 불과했던 AI 작성 코드 비율이, 2026년 5월 기준 Anthropic 코드베이스에 병합된 코드의 80% 이상을 Claude가 직접 작성하는 수준으로 급성장했다. 2026년 2분기 기준, 일반 엔지니어가 하루에 병합하는 코드량은 2024년 대비 8배에 달한다. 특히 주목할 만한 것은 재귀적 자기개선(Recursive Self-Improvement)의 초기 신호가 포착됐다는 점이다. 내부 최적화 벤치마크에서 Anthropic의 Mythos Preview 모델은 AI 훈련 코드를 52배 가속하는 성과를 냈으며, 숙련된 인간 개발자가 동일 코드베이스에서 최대 4배 속도 향상에 4~8시간을 쏟는 것과 극명히 대비된다. 기업이 이 패러다임을 따르려면 '개발자 보조' 모델에서 자동화 팩토리 아키텍처로의 전환이 필수적이며, 자동화된 코드베이스 80% 달성은 API 토큰 구매를 넘어 총체적인 문화 혁신과 개발자 역할 불안 완화 전략, 엄격한 자동화 검증 체계 구축을 요구한다.

Google은 2026년 6월 3일, 별도 인코더 네트워크 없이 텍스트·이미지·오디오·영상을 처리하며 16GB RAM 또는 VRAM이 탑재된 모든 노트북에서 실행되는 120억 파라미터 오픈 웨이트 모델 Gemma 4 12B를 Apache 2.0 라이선스로 무료 공개했다. 이 모델의 핵심 혁신은 인코더 프리(Encoder-Free) 통합 아키텍처다. 비전 처리는 단일 행렬 곱셈으로 구성된 경량 임베딩 모듈로 대체해 LLM 백본이 시각 정보를 직접 처리하도록 했고, 오디오 처리는 인코더를 완전히 제거해 원시 오디오 신호를 텍스트 토큰과 동일한 차원 공간으로 직접 투영하는 방식으로 단순화했다. 이 통합 아키텍처는 256,000 토큰 컨텍스트 윈도우를 지원해 약 200페이지 분량의 문서, 대용량 코드베이스, 또는 수 시간 분량의 오디오를 단일 패스로 처리할 수 있다. 의료·금융·국방 등 데이터 외부 전송이 불허되는 규제 산업과 온디바이스 AI 에이전트 워크플로에 특히 전략적 가치가 높다.

AI 에이전트가 만드는 데이터 사일로 문제, Microsoft의 IQ + Rayfin으로 통합 거버넌스 정면 돌파

엔터프라이즈 AI의 병목은 더 이상 모델 역량이 아니라 일관된 공유 데이터 컨텍스트의 부재이며, 새로운 에이전트마다 비즈니스 운영 방식·데이터 위치·준수 규칙을 매번 새로 학습해야 해 조율과 확장이 불가능하다. Microsoft는 Build 2026에서 이 문제를 정면 돌파했다. Microsoft IQ는 Work IQ(업무 방식 캡처), Fabric IQ(비즈니스 운영 모델링), Foundry IQ(지식 검색·재활용), Web IQ(실시간 웹 글로벌 컨텍스트)의 4가지 인텔리전스 레이어를 통합해 에이전트를 위한 공유 기반을 제공한다. 한편 Rayfin은 에이전트가 생성한 애플리케이션을 Microsoft Fabric에 직접 배포해 앱 데이터가 기본적으로 OneLake에 저장되고, 외부에 새 사일로를 만드는 대신 Microsoft IQ 컨텍스트 레이어로 피드백되도록 설계됐다. "IQ는 에이전트에게 알아야 할 지식을 주고, Rayfin은 안전하게 행동할 수 있는 표면을 제공한다"는 분석처럼, 두 시스템의 결합이 시스템 오브 레코드에 대한 에이전트의 쓰기 작업을 신뢰 가능하게 만드는 핵심 조합으로 평가된다.
